# Hotel Booking System (Python)

This is a simple console-based hotel booking system built using Python and CSV files as a database.

## Features

- View available hotels
- Check hotel availability
- Book a hotel
- Credit card validation
- Secure card authentication
- Optional SPA package
- Reservation ticket generation

## Technologies Used

- Python
- Pandas
- CSV files (used as a simple database)

## Project Structure

hotel-booking-system/

main.py - Basic hotel booking system  
main2.py - Hotel booking system with SPA package feature  
main3.py - Advanced OOP concepts demonstration  

hotels.csv - Hotel information database  
cards.csv - Credit card validation data  
card_security.csv - Credit card authentication passwords  

requirements.txt - Python dependencies  
README.md - Project documentation  

## How to Run the Project

1. Clone the repository

git clone https://github.com/your-username/hotel-booking-system.git

2. Navigate to the project folder

cd hotel-booking-system

3. Install required libraries

pip install -r requirements.txt

4. Run the program

python main.py

## Example Workflow

1. The system displays available hotels.
2. The user enters the hotel ID.
3. The system checks if the hotel is available.
4. Credit card details are validated.
5. Secure authentication is performed.
6. The hotel is booked and a reservation ticket is generated.
